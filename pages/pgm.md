* PPM (Portable PixMap)
* PGM (Portable GreyMap)
* PBM (Portable BitMap)

* See also [rgb](rgb)

```bash
(echo "P5 512 4096 255"; cat filename) > foo.pgm
```

```bash
(echo "P4 512 4096 255"; cat filename) > foo.pbm
```
