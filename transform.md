# actions and keyboard shortcuts
```js
actions.duplicateSelection()

actions.align(horizontal, vertical);
horizontal = vertical = -1 || 0 || 1 || false;
```

in hotkeys.js
```js

case 'd':
  if (commandKey) {
    dispatch(actions.duplicateSelection());
  }
  break;
```
